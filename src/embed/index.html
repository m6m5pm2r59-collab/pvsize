What these embeds are for

These are lightweight, editable PVSize calculators for partner sites. They are planning estimates only, and the inputs and assumptions stay visible so a visitor can challenge them.

PVSize does not promise installation, quotes, savings, or engineering approval through an embed.

Panel count embed

Use this for monthly usage and roof-fit estimates. It passes query parameters and UTM values through to the calculator.

Notes

If you need a taller frame, increase the iframe height. The embed pages will forward the current URL query string into the calculator so `utm_*`, `autocalc`, and other input parameters stay intact.
