1,500 kWh/month panel count

A high-usage home wants a first-pass panel count estimate before installer conversations.

Open editable calculator
Load
1,500 kWh/month, about 50 kWh/day
Solar input
4.5 sun hours, 400 W panels
Efficiency
80% simple system efficiency

Simplified approach: panel daily kWh = panel watts / 1000 * sun hours * system efficiency. Under these assumptions, the planning estimate is about 35 panels before roof layout, shading, and local design constraints.

Overnight outage battery sizing

A homeowner wants essential overnight backup, not a whole-home backup design.

Open editable calculator
Load
8 kWh essential overnight load
Backup
1 night planning window
Battery
LiFePO4, 90% DOD, 90% inverter efficiency

Simplified approach: required battery kWh = load kWh / (usable depth of discharge * inverter efficiency). Under these assumptions, the planning estimate is about 10 kWh nominal capacity.
