Off-Grid Solar System Sizing Guide

Off-grid solar needs a different sizing method because the grid is not there as backup. You need enough panels, usable battery capacity, inverter power, and backup margin for cloudy periods.

Short answer: start with daily kWh, then size battery days first and solar recharge second.

Core Sizing Inputs

  • Daily kWh for essential and normal loads.
  • Backup days for cloudy weather.
  • Battery usable capacity and depth of discharge.
  • Inverter size for peak appliance loads.
  • Generator or grid fallback, if any.
